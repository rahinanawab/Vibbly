# 🌐 Vibbly

**Vibbly** is a modern social media platform designed to connect users through posts, stories, real-time conversations, and interactive communities. The platform enables users to share multimedia content, build connections, participate in chat rooms, and engage with communities through a seamless social experience.

## ✨ Features

### 👤 User Management & Profiles
- User Registration & Authentication
- Profile Creation and Editing
- Profile Verification
- Username & Bio Customization
- Followers and Following Management
- Public User Profiles

### 📸 Posts & Stories
- Create and Share Posts
- Upload Photos and Videos
- Story Sharing
- Like and Comment on Content
- Video Views Tracking
- Media Preview Support

### 💬 Chat & Community Rooms
- One-to-One Messaging
- Group Chat Rooms
- Room Creation and Management
- Admin & Member Roles
- Room Join Requests
- Chat Notifications
- Message Management

### 🎙️ Audio Spaces
- Interactive Audio Communities
- User Participation Management
- Member Controls
- Real-Time Engagement

### 🔔 Notifications
- Follow Notifications
- Message Notifications
- Activity Updates
- Engagement Alerts

### 🔐 Privacy & Controls
- Follow / Unfollow Users
- Block and Manage Interactions
- Account Settings
- User Privacy Management

## 🚀 Why Vibbly?

Vibbly provides an interactive social networking experience where users can connect, communicate, and share moments through different forms of content. With multimedia sharing, real-time conversations, and community-based interactions, the platform creates an engaging environment for users to build meaningful connections.
